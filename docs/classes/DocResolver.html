    • DocResolver

    Class DocResolver<A, S, T>Internal

    Resolves Firestore document references and provides methods to navigate document relationships and collections.

    Example

    const resolver: DocResolver = ... // Create a resolver
    const parentResolver = resolver.parent();
    const doc = resolver(docRef);
    const colResolver = resolver.col('subcollection');

    Type Parameters

    • A extends AdaptorTypes

      The adaptor types used for Firestore operations.

    • S

      The source document data type.

    • T

      The target document data type after resolution..

    call

    call: (source: DocRef<A, S>) => Doc<A, T> = ...

    Allows the DocResolver to be invoked directly, which is the preferred method to resolve a document reference. Uses the resolve method of the current instance.

    Type declaration

      • (source: DocRef<A, S>): Doc<A, T>

    • col<C>(colPath: string): ColResolver<A, S, C>

      Resolves a collection reference from the given document reference and returns a new ColResolver instance. This allows for further querying or manipulation of the collection.

      Type Parameters

      • C

      Parameters

      • colPath: string

        The path to the subcollection.

      Returns ColResolver<A, S, C>

      A new ColResolver instance associated with the resolved collection reference.

      Example

      const postsResolver = accountResolver.col('posts');
      const posts = await colResolver(docRef).get();

      ): QueryResolver<A, S, C>

      Extends the current document reference to a new QueryResolver instance. Allowing further extension or resolving of the query from the source document. Construction of the query depends on the underlying adaptor.

      Type Parameters

      • C

      Parameters

      • getQuery: (source: DocRef<A, T>) => QueryRef<A, C>

        A function that receives the current document reference and returns a new query reference with additional query constraints.

      Returns QueryResolver<A, S, C>

      A new QueryResolver instance configured with the transformed query.

      Remarks

      This could be particularly useful when used with a Firestore 'collectionGroup' query or other pre-defined QueryResolvers

      Example

      const postsResolver = accountResolver.query(
       (account) => newQuery('posts').where('accountRef', '==', account)
