    Interface IDocResolver<A, S, T>Internal

    Resolves document references and provides methods to navigate document relationships and collections.

    Example

    const resolver: IDocResolver = ... // Create a resolver
    const parentResolver = resolver.parent();
    const doc = resolver(docRef);
    const colResolver = resolver.col('subcollection');

    col

    • col<C>(colPath: string): IColResolver<A, S, C>

      Resolves a collection reference from the given document reference and returns a new IColResolver. This allows for further querying or manipulation of the collection.

      Type Parameters

      • C

      Parameters

      • colPath: string

        The path to the subcollection.

      Returns IColResolver<A, S, C>

      A new IColResolver associated with the resolved collection reference.

      Example

      const accountResolver: IDocResolver<Types, IUser, IAccount>;
      const postsResolver = accountResolver.col('posts');
      const posts = await colResolver(docRef).get();

    parent

    • parent<P>(): IDocResolver<A, S, P>

      Creates a new IDocResolver instance that resolves the parent document reference of the current document. This is useful for traversing up one level in a document hierarchy.

      Type Parameters

      • P

        The type of the parent document's data.

      Returns IDocResolver<A, S, P>

      A new IDocResolver configured to resolve the parent document reference.

      Example

      const parentResolver = resolver.parent<ParentType>();
      const parentDoc = await parentResolver(docRef).get();

    • query<C>(
          getQuery: (source: DocRef<A, T>) => QueryRef<A, C>,
      ): IQueryResolver<A, S, C>

      Extends the current document reference to a new IQueryResolver. Allowing further extension or resolving of the query from the source document. Construction of the query depends on the underlying adaptor.

      Type Parameters

      • C

      Parameters

      • getQuery: (source: DocRef<A, T>) => QueryRef<A, C>

        A function that receives the current document reference and returns a new query reference with additional query constraints.

      Returns IQueryResolver<A, S, C>

      A new IQueryResolver configured with the transformed query.

      Remarks

      This could be particularly useful when used with a Firestore 'collectionGroup' query or other pre-defined IQueryResolvers

      Example

      const accountResolver: IDocResolver<Types, IUser, IAccount>;
      const postsResolver = accountResolver.query(
       (account) => newQuery('posts').where('accountRef', '==', account)
      );

    resolve

    • resolve(source: DocRef<A, S>): IDoc<A, T>

      Resolves a document reference from the given document reference and returns a new IDoc. However, the preferred method to do this is by invoking the IDocResolver directly.

      Parameters

      • source: DocRef<A, S>

        The document reference to resolve.

      Returns IDoc<A, T>

      A new IDoc corresponding to the resolved reference.

      Example

      const data = await resolver(docRef).get() // Called directly (preferred)
      const data = await resolver.call(docRef).get() // Called as a function
